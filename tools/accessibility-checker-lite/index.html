    How it works

    This lightweight checker parses your HTML in-browser and runs best-effort checks: WCAG AA text contrast (vs assumed white if unknown), heading outline, image alt, generic link text, form labeling, and recommended landmarks (<main>, <nav>, <header>, <footer>). No data leaves your device.

    FAQs

    Is this a full audit?

    No—this is an MVP pre-check. For production, pair it with manual testing, keyboard review, and screen reader checks.

    What contrast does it use?

    WCAG 2.1 AA threshold for normal text (4.5:1). It compares inline colors (or simple attributes) against white when a background can’t be determined.
