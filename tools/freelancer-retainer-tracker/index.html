How it works

Enter your client retainer details and track hours and payment status. Your data is saved locally in your browser using localStorage — nothing is uploaded or shared.

  • Add rows with the form above.
  • Double-click table cells to edit inline.
  • Filter for Active or Pending only.
  • Export the visible table as CSV anytime.

FAQs

Is this tool free?

Yes. It runs entirely in your browser — no login, no backend.

What does “Monthly Total” include?

It sums the Retainer Amount for all visible rows. “Pending Payments” sums retainers with status set to Pending. “Avg. Hourly Rate” averages the provided hourly rates across visible rows.
