How it works

This tool converts PNG and JPEG images to modern WebP directly in your browser using HTML Canvas. No uploads, no tracking. Adjust quality, optionally resize, and download optimized images with preserved transparency.

FAQs

Is this really client-side only?

Yes. Everything runs in your browser. Files never leave your device.

Does WebP support transparency?

Yes. WebP supports alpha, so PNG transparency is preserved automatically.
